Template Options

You can use book information in the note title and content templates - parameter names are surrounded with double curly braces - for example, {{ short title }}. The preview pane will show you how your template will render with an example book.

Parameter Name Description Example Output
full_title The full title of the book. Note that this may include colons, which aren't legal note name in Obsidian (use short_title instead) {{ full_title }}
short_title the title of the book up to the first colon {{ short_title }}
publication_year the publication year of the book's first edition {{ publication_year }}
formatted_authors a comma separated list of authors, surrounded with Obsidian link notation {{ formatted_authors }}
cover_image_url a URL for the main Goodreads cover image {{ cover_image_url }}
cover_image same as the above URL, but embedded as a Markdown image {{ cover_image }}
abstract the book's description from the page {{ abstract }}
series the book's series data {{ series }}
date Displays the current date and time according to the specified format. Defaults to YYYY-MM-DD if no format is specified. Specify a format with a colon after the variable name. {{ date }} → {{ date }}
{{ date:YYYY-MM }} → {{ date:YYYY-MM }}
{{ date:dddd MMM do, Y }} → {{ date:dddd MMM do, Y }}
{{ date:HH:mm }} → {{ date:HH:mm }}
time same as date, but defaults to HH:mm {{ time }}
